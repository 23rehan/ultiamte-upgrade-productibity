## ⚡ Installation

1. **Clone the repository**  
   ```sh
   git clone https://github.com/your-username/ultimate-productivity.git
   ```
2. **Open the project folder**  
   ```sh
   cd ultimate-productivity
   ```
3. **Open `index.html` in your browser**  
   No extra setup needed! 🎉
